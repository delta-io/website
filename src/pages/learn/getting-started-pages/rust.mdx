### Getting Started with Delta Lake

<hr />

1. Open a bash shell (if on windows use git bash, WSL, or any shell configured for bash commands)

2. Run a container from the image with a bash entrypoint ([DockerHub](/learn/docker-pages/docker-hub) | [build](/learn/docker-pages/docker-build))

3. Execute `examples/read_delta_table.rs` to review the Delta Lake table metadata and files of the `covid19_nyt` Delta Lake table.

   <CodeTabs>

   ```bash
   cd rs
   cargo run --example read_delta_table
   ```

   </CodeTabs>

   <Info title="Note" level="info">
     If using [Delta Lake DockerHub](https://go.delta.io/dockerhub), sometimes
     the Rust environment hasn't been configured. To resolve this, run the
     command `source "$HOME/.cargo/env"`
   </Info>
